# Chefs & Gastronomie — Abidjan

Site vitrine du restaurant **Chefs & Gastronomie**, expérience gastronomique privée à Abidjan : un voyage culinaire sans carte, guidé par le Chef, sur mesure et sécurisé. Sur réservation uniquement.

Site statique (HTML / CSS / JS), hébergé via GitHub Pages.

## Contenu
- `index.html` — page unique avec effets de scroll
- `styles.css` — design « ivoire & lumière »
- `script.js` — animations au défilement, parallaxe, menu mobile
- `assets/` — images et vidéos
